# Фоны уровней (необязательно)

Сюда можно положить картинки-фоны для конкретных уровней.

## Как это работает
- Имя файла: `level_01.png`, `level_02.png`, `level_03.png` и т.д.
  (число = номер уровня, начиная с 1).
- Если для уровня есть такая картинка — игра покажет ЕЁ вместо
  цветного градиента.
- Если картинки нет — берётся градиент из `settings.py`
  (список `LEVEL_BACKGROUNDS`).

## Размер
Любой PNG автоматически растянется до внутреннего разрешения экрана
(480 × 288 пикселей). Чтобы фон был чётким, рисуй примерно в этом
размере или кратном ему (например 960 × 576).

## Совет
Фон рисуется ПОД игровым полем и не двигается вместе с миром
(только лёгкий параллакс из точек). Поэтому делай его спокойным,
без важных деталей в центре — иначе он будет спорить с геймплеем.
