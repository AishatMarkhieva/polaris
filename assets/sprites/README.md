# Свои спрайты

Игра работает БЕЗ картинок — все спрайты рисуются кодом. Но как только ты
положишь сюда PNG с нужным именем, игра автоматически возьмёт ТВОЙ спрайт.

Имена файлов (всё опционально, клади только то, что нарисовал):

| Файл                | Что это             | Рекомендуемый размер |
|---------------------|---------------------|----------------------|
| player_blue.png     | герой, синий цвет   | 12 × 14 px           |
| player_orange.png   | герой, оранжевый    | 12 × 14 px           |
| wall.png            | стена (#)           | 16 × 16 px           |
| block_blue.png      | синий блок (B)      | 16 × 16 px           |
| block_orange.png    | оранжевый блок (O)  | 16 × 16 px           |
| spike.png           | шипы (^)            | 16 × 16 px           |
| crystal.png         | кристалл (*)        | 16 × 16 px           |
| exit.png            | выход (E)           | 16 × 16 px           |

Любой PNG автоматически масштабируется под нужный размер, но лучше рисовать
сразу в этих размерах — так пиксель-арт будет чётким. Формат — PNG с
прозрачностью (альфа-каналом).
